## Barefoot Frontend (TEAM BRAVO)

[![Build Status](https://app.travis-ci.com/atlp-rwanda/bravo-bn-fe.svg?branch=develop)](https://app.travis-ci.com/github/atlp-rwanda/bravo-bn-fe)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
